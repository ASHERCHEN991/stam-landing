# כתר הסופר — דף נחיתה

דף נחיתה סטטי ורספונסיבי לשירותי סת״ם. אין צורך בהתקנה או בתהליך build.

## צפייה מקומית

```bash
python3 -m http.server 4173
```

לאחר מכן פתחו בדפדפן את [http://localhost:4173](http://localhost:4173).

## GitHub Pages

הדף משתמש בנתיבים יחסיים לכל נכסי האתר המקומיים (`./src/` ו-`./assets/`), ולכן עובד גם כאשר הוא מתפרסם תחת נתיב המאגר:

`https://<username>.github.io/stam-landing/`

## התאמות לפני פרסום

חפשו ב-`index.html` את ערכי ה-placeholder הבאים והחליפו אותם:

- `כתר הסופר` — שם העסק.
- `+000 000 000 000` ו-`000000000000` — מספר הטלפון וקישור ה-WhatsApp.
- `https://www.youtube.com/@placeholder` — קישור לערוץ ה-YouTube.
- `[שם מלא]`, `[עיר / מדינה]` — פרטי ההמלצה.
